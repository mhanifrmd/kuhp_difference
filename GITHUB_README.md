# KUHP Analyzer

AI-powered application for analyzing differences between old and new Indonesian Criminal Code (KUHP) using Gemini 2.0 Flash.

## Repository Structure
This repository contains a complete web application with:
- FastAPI backend with Gemini 2.0 Flash integration
- Next.js frontend with modern UI
- Docker containerization
- Google Cloud Run deployment ready
- Comprehensive documentation

## Quick Start
1. Clone this repository
2. Follow the setup instructions in README.md
3. Deploy to Google Cloud Run using the provided scripts

## Features
- AI-powered legal document analysis
- Clean, emoji-free codebase
- Production-ready deployment configuration
- Comprehensive testing and verification tools

For detailed instructions, see README.md
